# KrishiLink - Smart Farming & Crop Management Platform

KrishiLink is a full-stack web application designed to connect farmers, buyers, and agricultural enthusiasts.  
It enables users to post, view, and manage crops; express buying interest; and streamline the communication between producers and buyers.

# Live URL
**Frontend (Live Site):** [Click here to visit KrishiLink Project](https://krishilink.web.app)

## Tech Stack

### Frontend
- **React 19** — Modern UI library for building responsive interfaces  
- **React Router 7** — Declarative routing and data loading  
- **Tailwind CSS 4** — Utility-first styling for rapid UI development  
- **DaisyUI** — Beautiful Tailwind UI components  
- **Framer Motion** — Smooth animations and transitions  
- **Firebase** — Authentication & hosting  
- **Axios** — Simplified HTTP requests  
- **React Toastify** & **SweetAlert2** — Elegant notifications and alerts  
- **Lucide React Icons** — Clean icon library

### Backend
- **Node.js + Express 5** — RESTful API with structured routes  
- **MongoDB (Atlas)** — NoSQL database for crop and user data  
- **Dotenv** — Environment variable management  
- **CORS** — Cross-origin resource sharing support  

## Key Features

### Farmers
- Add, edit, and delete crop listings  
- View all posted crops  
- Manage buyer interests for their crops  

### Buyers
- Browse all available crops  
- View detailed crop information  
- Express interest in purchasing crops  
- Track interest status (pending / accepted / rejected)  

### Authentication
- Firebase-based login, registration, and password reset  
- Protected routes for logged-in users  

### Extra Features
- Real-time search filtering on the frontend  
- Responsive, mobile-friendly UI  
- Smooth animations with Framer Motion  
- Toast & SweetAlert feedback system
